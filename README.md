# Python Telegram Bot Replit

Fork the Code
Set up `TELEGRAM_BOT_TOKEN` and `BOT_USERNAME` in the `secrets` and run it

## Local
Create `.env` file

```shell
export TELEGRAM_BOT_TOKEN=""
export BOT_USERNAME=""
```


```bash
pip install poetry # download poetry, a dependency manager

source .env
poetry install
poetry run python main.py
```
